# ResearchOS MVP

ResearchOS is an AI-powered IRB amendment drafting tool for university research labs. Researchers describe a protocol change in plain English, upload their approved IRB protocol PDF, and receive a complete, submission-ready amendment document in under 90 seconds. Built for UMBC and other institutions using Kuali Research.

> **Important:** The app (`/app`) must be accessed via the FastAPI server at `http://localhost:8000/app` — never opened as a local file. Fetch calls use relative paths that require the server origin.

## Setup

```bash
# 1. Clone and install
cd researchos
pip install -r requirements.txt

# 2. Set API key
cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY
# Get a key at: https://console.anthropic.com

# 3. Run
uvicorn backend.main:app --reload --port 8000

# 4. Open
# Landing page: http://localhost:8000
# App:          http://localhost:8000/app
# Health:       http://localhost:8000/api/health
```

## Tech stack

- **Backend:** FastAPI + Python 3.12, uvicorn
- **AI:** Anthropic Claude API (`claude-sonnet-4-20250514`) — 3-stage pipeline
- **PDF:** PyMuPDF (fitz) for text extraction
- **Documents:** python-docx for Word generation
- **Frontend:** Vanilla HTML/CSS/JS — no frameworks, no build step

## Important disclaimer

All outputs generated by ResearchOS are **AI-generated drafts for researcher review only**. The Principal Investigator must review and certify all content before submission to the IRB. ResearchOS does not guarantee IRB approval. Items marked `[RESEARCHER TO CONFIRM]` require manual input before submission.
